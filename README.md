## Introduction

A starter [Vite](https://vitejs.dev/) template having:

- React
- TypeScript
- Vite
- Tailwind CSS
- Multiple eslint & prettier plugins installed

## Why use this template?

I am using this template because it contains technologies I have used in the past and am comfortable with using.

- Vite
    - Vite is a fast front-end build tool that is designed to optimize the development experience.

- TypeScript
    - Superset of JavaScript to catch potential errors during development, leading to more robust and maintainable code

- Tailwind CSS
    - CSS framework that provides a set of pre-designed utility classes, allowing for rapid and consistent styling

- Eslint
    - To identify and fix common issues in the TypeScript code

- Prettier
    - Code formatter that helps maintain consistent coding styles across the project
